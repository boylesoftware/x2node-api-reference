Class: ServiceCall

x2node-ws~ ServiceCall

Web-service call. An instance is created automatically by the framework for each and every web-service call and provides context for all the components that participate in the call handling (such as call handlers).

new ServiceCall(httpRequest, httpResponse)

Note: The constructor is not accessible from the client code. Instances are created internally by the framework.
Parameters:
Name Type Description
httpRequest http.external:IncomingMessage HTTP request.
httpResponse http.external:ServerResponse HTTP response.

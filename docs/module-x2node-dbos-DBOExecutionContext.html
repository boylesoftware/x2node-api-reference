Class: DBOExecutionContext

x2node-dbos~ DBOExecutionContext

Base class for various DBO execution contexts.

new DBOExecutionContext(dbo, txOrCon, actor [, filterParams] [, entangledUpdates])

Note: The constructor is not accessible from the client code. Instances are provided to the application components by the framework where needed.
Parameters:
Name Type Argument Description
dbo module:x2node-dbos~AbstractDBO The DBO.
txOrCon module:x2node-dbos~Transaction | * Active transaction or database connection.

 Actor executing the command.
filterParams Object.<string, *> <optional>
 Optional filter parameters passed to the DBO execution.
entangledUpdates Object.<string, Set.<(string|number)>> <optional>
 Optional entangled updates object.

If transaction was provided but it was not active.
